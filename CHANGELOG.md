# Changelog

All notable changes to `matowens/webpage-monitor` will be documented in this file.

## [1.0.0] - 2026-07-12

Initial public release.

- standalone Laravel package structure with auto-discovery
- package-owned Pest and Orchestra Testbench test harness
- one-off webpage checks for availability, contains, and selector extraction
- saved monitors with manual and scheduled execution
- selector normalization, hashing, and change detection
- database-backed scheduling and durable claim handling
- state-aware mail notifications for monitor transitions and lifecycle events
